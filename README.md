# Portfolio with Analytics

This project is a React-based portfolio website with built-in visitor analytics using Supabase.

## Features
- **Modern UI**: Built with React, Tailwind CSS, and Framer Motion.
- **Analytics**: Tracks visitor counts and distinguishes between humans and bots.
- **Supabase Integration**: Stores visit data in a real-time database.

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Variables**:
    Ensure you have a `.env` file in the root directory with your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_project_url
    VITE_SUPABASE_ANON_KEY=your_anon_key
    ```

3.  **Database Setup**:
    Run the following SQL in your Supabase SQL Editor to create the required table:
    ```sql
    create table visits (
      id bigint generated by default as identity primary key,
      created_at timestamp with time zone default timezone('utc'::text, now()) not null,
      user_agent text,
      is_bot boolean default false,
      path text
    );
    ```

4.  **Run Locally**:
    ```bash
    npm run dev
    ```

5.  **Build for Production**:
    ```bash
    npm run build
    ```
